# hare-simple
## 系统说明
    hare后台管理系统系列中最轻量的一个版本，前后端分离开发，意在精简、轻便，
    使用Spring Boot + Mybatis Plus  + Spring Security + JWT适合作
    为后台管理系统的轻量级版本。

## 核心依赖

| 项目                     | 版本    |
|------------------------|---------|
| Spring Boot            | 2.7.18  |
| Mysql                  | 8.2.0   |
| Mybatis Plus           | 3.5.3.2 |
| Oauth2 Resource Server | 2.7.18  |
